# dotfiles

## Install

```sh
git clone https://github.com/ansanloms/dotfiles.git
cd dotfiles
```

### macOS / Linux

```sh
bash install.sh
```

### Windows

on Command Prompt:

```bat
powershell -NoProfile -ExecutionPolicy Unrestricted ./install.ps1
```

on Powershell:

```powershell
.\install.ps1
```
